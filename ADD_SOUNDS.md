# How to Add Sound Files

## Method 1: Using Finder (macOS)

1. Open Finder
2. Navigate to your app folder: `/Users/faiyaad/coding apps/focus app/`
3. Double-click the `sounds` folder to open it
4. Download or drag your MP3 files into this folder
5. Make sure the files are named exactly:
   - `rain.mp3`
   - `ocean.mp3`
   - `fire.mp3`
   - `wind.mp3`
   - `forest.mp3`
   - `cafe.mp3`
   - `cityscape.mp3`

## Method 2: Using Terminal

Open Terminal and run these commands:

```bash
cd "/Users/faiyaad/coding apps/focus app/sounds"

# Download files from the internet (example - you'll need to find your own URLs):
# curl -o rain.mp3 "YOUR_DOWNLOAD_URL_HERE"
```

## Method 3: Drag and Drop

1. Download the MP3 files to your Downloads folder
2. Open Finder
3. Go to: `/Users/faiyaad/coding apps/focus app/sounds`
4. Drag the files from Downloads into the sounds folder
5. Rename them to match the required names above

## Where to Get Free Sounds:

- **Freesound.org**: https://freesound.org (search for "rain ambient", "ocean waves", etc.)
- **Pixabay**: https://pixabay.com/music/search/ambient/
- **Zapsplat**: https://www.zapsplat.com
- **Mixkit**: https://mixkit.co/free-sound-effects/

**Important**: Make sure the sounds loop seamlessly (they should sound good when repeated)
